# Virat-Kohli-Century-Analytics-Dashboard

This project analyzes the centuries scored by Virat Kohli using an interactive Power BI dashboard. The goal of this project is to transform raw cricket match data into meaningful visual insights and explore performance trends across formats, venues, and opponents.

## Features

- Interactive visualizations of Virat Kohli's century performances
- Analysis across different cricket formats (Test, ODI, T20)
- Performance trends by venue and opponent
- Key statistics and milestones
- User-friendly dashboard interface

## Analysis Questions

The dashboard addresses the following key analysis questions:

### Format-Based Analysis
- How are Virat Kohli's centuries distributed across Test, ODI, and T20 formats?
- In which format has he scored the highest number of centuries?
- Which format contributes the largest share of his total centuries?

### Opposition Analysis
- Against which teams has Virat Kohli scored the most centuries?
- Which opponents appear most frequently in his century list?
- Are there specific teams against whom he consistently performs well?

### Venue Analysis
- At which venues or stadiums has Virat Kohli scored the most centuries?
- Do certain venues appear repeatedly in his century records?
- Is there a noticeable pattern in home vs away performances?

### Time-Based Trends
- How has Virat Kohli's century-scoring trend changed over the years?
- Which year had the highest number of centuries?
- Are there periods where his performance peaked significantly?

### Performance Distribution
- What percentage of his centuries come from each cricket format?
- How evenly are his centuries distributed across different opponents?

### Overall Performance Insights
- What patterns can be identified in Virat Kohli's century-scoring performances?
- How can data visualization help reveal insights about his consistency and dominance in cricket?

## Requirements

- Power BI Desktop (latest version recommended)
- Windows OS (for Power BI Desktop)

## Installation

1. Clone this repository:
   ```
   git clone https://github.com/yourusername/Virat-Kohli-Century-Analytics-Dashboard.git
   ```

2. Download and install Power BI Desktop from [Microsoft's official website](https://powerbi.microsoft.com/desktop/)

3. Open the `virat.pbix` file located in the `Dashboard File/` folder using Power BI Desktop

## Usage

1. Open the `virat.pbix` file in Power BI Desktop
2. Interact with the visualizations using filters and slicers
3. Explore different tabs for various analyses:
   - Overview of centuries
   - Performance by format
   - Venue analysis
   - Opponent analysis
